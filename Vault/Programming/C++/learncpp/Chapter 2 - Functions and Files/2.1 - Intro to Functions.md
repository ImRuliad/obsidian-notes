*A Function is a reusable sequence of statements designed to do a particular job.*

---

# <u> User Defined Functions </u>

*Example of a user-defined function*
```cpp
returnType functionName(){  //This is the function header.
	//This is the function body
}
```

> [!Danger] A Function must be defined before it is called.

```cpp
#include iostream

void doPrint() {
	std::cout << "In doPrint()\n";
}

void main() {
	std::cout << "Starting doPrint()\n";
	doPrint()
	std::cout << "Ending doPrint()\n";
	
	return 0;
}
// This produces the output:
// Starting doPrint()
// In doPrint()
// Ending doPrint()
```

---

# <u> Functions That Call Functions That Call Functions </u>
*`main()` can call other functions. The functions called by `main()` can also call other functions (and those functions can call functions too, etc…). In the following program, function `main()` calls function `doA()`, which calls function `doB()`:*

```cpp
#include iostream

void doB() {
	std::cout << "In doB()\n";
}

void doA() {
	std::cout << "Starting doA()\n";
	doB();
	std::cout << "Ending doA()\n";
}

void main() {
	std::cout << "Starting main()\n";
	doA();
	std::cout << "Ending main()\n";
	
	return 0;
}
// Starting main()
// Starting doA()
// In doB()
// Ending doA()
// Ending main()
```

> [!Danger] Nested functions are NOT supported
