
## This is my first attempt at a solution:
```python
class Solution:
	def maxDepth(self, root: Optional[TreeNode]) -> int:
		max_depth = [0]
		def dfs(root):
			if not root: return 0
			left = dfs(root.left)
			right = dfs(root.right)
			max_depth[0] = max(left, right)
	dfs(root)
	return max_depth
```

## The issues here is that:
- Your `dfs` never returns a depth for non-null nodes. After computing `left` and `right`, you must `return 1 + max(left, right)` so callers get a number.
- You’re assigning `max_depth[0] = max(left, right)`, which (a) omits the `+ 1` for the current node and (b) isn’t even needed if you just return depth.
- At the end you return the list `max_depth`, not an `int`. Even if you kept the “single-slot box,” you’d need to return `max_depth[0]`.


## This is the correct solution:
```python
class Solution:
	def maxDepth(self, root: Optional[TreeNode]) -> int:
		max_depth = [0]
		def dfs(root):
			if not root: return 0
			left = dfs(root.left)
			right = dfs(root.right)
			max_depth[0] = 1 + max(left, right)
			return max_depth[0]
	dfs(root)
	return max_depth[0]
```
